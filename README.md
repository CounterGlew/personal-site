# Zheng Meihao Personal Site

This repository contains a static personal website built from resume content.

## Local preview

Open `index.html` directly in a browser, or serve the directory with any static file server.

## GitHub Pages

The site is configured for GitHub Pages via GitHub Actions.

After the repository is pushed to GitHub:

1. Ensure Pages is enabled to deploy from GitHub Actions.
2. Push to the default branch.
3. GitHub Actions will publish the site automatically.
